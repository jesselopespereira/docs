## Visão Geral

A Kriging API possui um único endpoint que recebe pontos geográficos com valores e retorna uma imagem PNG com o mapa de calor interpolado.

<Note>
  A API está disponível em: `http://localhost:3000/api/kriging`
</Note>

## Faça sua primeira requisição

### Requisição básica

Aqui está um exemplo mínimo para gerar um mapa de calor:

```bash
curl -X POST http://localhost:3000/api/kriging \
  -H "Content-Type: application/json" \
  -d '{
    "points": [
      { "x": -46.6333, "y": -23.5505, "value": 25.5 },
      { "x": -46.6400, "y": -23.5600, "value": 28.3 },
      { "x": -46.6250, "y": -23.5450, "value": 22.1 }
    ]
  }'
```

### Resposta

```json
{
  "imageUrl": "http://localhost:3000/public/kriging_abc123def456.png"
}
```

A API retorna uma URL para a imagem PNG gerada que você pode usar diretamente.

## Exemplos Práticos

### Com cores customizadas

Defina um gradiente de cores personalizado para seu mapa:

```javascript
const response = await fetch('http://localhost:3000/api/kriging', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    points: [
      { x: -46.6333, y: -23.5505, value: 10 },
      { x: -46.6400, y: -23.5600, value: 50 },
      { x: -46.6250, y: -23.5450, value: 30 }
    ],
    colors: ['#0000FF', '#00FF00', '#FFFF00', '#FF0000'],
    width: 800
  })
});

const { imageUrl } = await response.json();
```

### Com delimitação por polígono

Use um polígono GeoJSON para delimitar a área de interpolação:

```python
import requests

payload = {
    "points": [
        {"x": -46.6333, "y": -23.5505, "value": 25.5},
        {"x": -46.6400, "y": -23.5600, "value": 28.3},
        {"x": -46.6250, "y": -23.5450, "value": 22.1}
    ],
    "polygon": {
        "type": "Polygon",
        "coordinates": [[
            [-46.650, -23.560],
            [-46.650, -23.540],
            [-46.620, -23.540],
            [-46.620, -23.560],
            [-46.650, -23.560]
        ]]
    },
    "width": 600
}

response = requests.post('http://localhost:3000/api/kriging', json=payload)
result = response.json()
print(result['imageUrl'])
```

## Parâmetros

<ParamField path="points" type="array" required>
  Array de pontos com coordenadas e valores. Mínimo de 3 pontos necessários.
  
  <Expandable title="Estrutura do ponto">
    <ResponseField name="x" type="number" required>
      Longitude do ponto
    </ResponseField>
    <ResponseField name="y" type="number" required>
      Latitude do ponto
    </ResponseField>
    <ResponseField name="value" type="number" required>
      Valor a ser interpolado neste ponto
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField path="colors" type="array">
  Array de cores em formato hexadecimal para o gradiente. Se omitido, usa paleta padrão de verde para vermelho.
</ParamField>

<ParamField path="width" type="number" default="400">
  Largura da imagem em pixels. A imagem sempre será quadrada.
</ParamField>

<ParamField path="polygon" type="object">
  Polígono GeoJSON para delimitar a área de interpolação. Deve ser um Polygon válido com coordinates.
</ParamField>
