## Overview

Krigg has a single endpoint that receives geographic points with values ​​and returns a PNG image with the interpolated heat map.

<Note>
  The API is available at: `http://localhost:3000/api/kriging`
</Note>

## Place your first request

### Basic requirement

Here's a minimal example for generating a heat map:

```bash
curl -X POST http://localhost:3000/api/kriging \
  -H "Content-Type: application/json" \
  -d '{
    "points": [
      { "x": -46.6333, "y": -23.5505, "value": 25.5 },
      { "x": -46.6400, "y": -23.5600, "value": 28.3 },
      { "x": -46.6250, "y": -23.5450, "value": 22.1 }
    ]
  }'
```

### Response

```json
{
  "imageUrl": "http://localhost:3000/public/kriging_abc123def456.png"
}
```

The API returns a URL to the generated PNG image that you can use directly.

## Practical Examples

### With custom colors

Define a custom color gradient for your map:

```javascript
const response = await fetch('http://localhost:3000/api/kriging', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    points: [
      { x: -46.6333, y: -23.5505, value: 10 },
      { x: -46.6400, y: -23.5600, value: 50 },
      { x: -46.6250, y: -23.5450, value: 30 }
    ],
    colors: ['#0000FF', '#00FF00', '#FFFF00', '#FF0000']
  })
});

const { imageUrl } = await response.json();
```

### With polygon delimitation

Use a GeoJSON polygon to delineate the interpolation area.:

```python
import requests

payload = {
    "points": [
        {"x": -46.6333, "y": -23.5505, "value": 25.5},
        {"x": -46.6400, "y": -23.5600, "value": 28.3},
        {"x": -46.6250, "y": -23.5450, "value": 22.1}
    ],
    "polygon": {
        "type": "Polygon",
        "coordinates": [[
            [-46.650, -23.560],
            [-46.650, -23.540],
            [-46.620, -23.540],
            [-46.620, -23.560],
            [-46.650, -23.560]
        ]]
    }
}

response = requests.post('http://localhost:3000/api/kriging', json=payload)
result = response.json()
print(result['imageUrl'])
```

## Parameters

<ParamField path="points" type="array" required>
  Array of points with coordinates and values. Minimum of 3 points required.
  
  <Expandable title="Point structure">
    <ResponseField name="x" type="number" required>
      Longitude of the point
    </ResponseField>
    <ResponseField name="y" type="number" required>
      Latitude of the point
    </ResponseField>
    <ResponseField name="value" type="number" required>
      Value to be interpolated at this point
    </ResponseField>
  </Expandable>
</ParamField>

<ParamField path="colors" type="array">
  A hexadecimal color array for the gradient. If omitted, it uses the default green-to-red color palette.
</ParamField>

<ParamField path="polygon" type="object">
  GeoJSON polygon to delimit the interpolation area. It must be a valid polygon with coordinates.
</ParamField>
