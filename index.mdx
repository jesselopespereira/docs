## O que é a Kriging API?

A Kriging API é um serviço que permite gerar mapas de calor (heatmaps) através da interpolação espacial usando o método de Kriging. Ideal para visualização de dados geoespaciais como temperatura, poluição, concentração de substâncias, e qualquer outro fenômeno que varie espacialmente.

## O que é Kriging?

Kriging é um método de interpolação geoestatística que estima valores em locais não amostrados com base em valores observados em pontos conhecidos. É amplamente utilizado em:

- **Análise ambiental**: Mapas de temperatura, umidade, poluição
- **Agricultura de precisão**: Variação de nutrientes no solo
- **Geologia**: Análise de concentrações minerais
- **Meteorologia**: Previsões espaciais de fenômenos climáticos
